# Agent Roles Are Interchangeable

The agents included in this repository represent **roles within my personal workflow**, not recommendations for specific AI models.

The names, personalities and responsibilities assigned to each agent reflect the way I currently prefer to work while developing my own projects.

They are examples—not requirements.

If another model performs better as an Architect, Auditor, Archivist or Explorer, simply replace the agent while keeping the role.

The workflow is designed around **responsibilities**, not around products, companies or AI providers.

As models evolve, new agents will appear and others will disappear.

The roles should remain stable.

Feel free to rename them, merge them, split them or replace them entirely if doing so better suits your own workflow.

There is no "correct" combination.

The best workflow is the one that helps **you** think more clearly, make better decisions and maintain control over your project.

---

> **MetalLabs Framework documents one possible way of working—not the only one.**
